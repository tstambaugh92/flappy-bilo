# Flappy Bilo

Flappy Bilo is a work-in-progress arcade game built with Godot. Guide Bilo
through incoming water bear obstacles without colliding with them.

## Requirements

- Godot Engine 4.6 or newer
- Git, if cloning the repository

## Running the Project

1. Clone or download this repository.
2. Open Godot's Project Manager.
3. Select **Import**, then choose this repository's `project.godot` file.
4. Open the project.
5. Press **F6** to run the current scene or **F5** to run the game.

## Controls

- **Space** or **left mouse button**: Jump
- **F**: Toggle fullscreen
